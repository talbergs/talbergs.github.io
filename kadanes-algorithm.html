Question:

A popular "Maximum Subarray Problem"

You are given some array of integer values, find greatest sum by summing up some subarray.

[1, 2, 3, -3, -9, 19, 2, 1, 0, -1, 4, -2]

The main complication in this problem comes with negative numbers.

Solution in bash below.

max_ending=$1
max_so_far=$1
shift

for i in ${@}
do
    max_ending=$(( $i > $max_ending + $i ? $i : $max_ending + $i ))
    max_so_far=$(( $max_so_far > $max_ending ? $max_so_far : $max_ending ))
done

echo $max_so_far
